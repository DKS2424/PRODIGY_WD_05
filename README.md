_______________________________________________________________________________________Weather App___________________________________________________________________________________________
This web application provides real-time weather forecasts based on the city name entered by the user. It displays essential weather information such as temperature, wind speed, and humidity.

Features:-
Search by City Name: Get weather updates by typing the name of any city.
Weather Details: Displays current temperature, wind speed, and humidity.
Dynamic and Interactive: Provides a user-friendly interface for a seamless experience.

Technologies Used:-
HTML
CSS
JavaScript
OpenWeather API


How to Use:-
Clone or download the repository.
Obtain an API key from OpenWeather and replace it in the JavaScript file.
Open the index.html file in any web browser.
Type the name of a city in the search box to get the current weather information.
